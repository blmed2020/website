    Was ist HTML

    HTML5 ist die fünfte Version von Hypertext Markup Language, einer Beschreibungssprache auf, die fast jede Website im World Wide Web aufgebaut ist. HTML erlaubt es Mediamatiker, Informatiker und Programmierer Webseiten mit Informationen zu füllen. Jedoch nicht diese zu gestalten. Dafür ist das CSS verantwortlich, worauf wir später zurückkommen werden. Erfunden wurde diese Sprache am 13. März 1989 von Tim Berners-Lee am CERN in Genf, sie war zur Strukturierung und Darstellung von wissenschaftlichen Informationen im Internet gedacht. Jedes HTML-File braucht ein HTML5 Grundgerüst (Unten abgebildet), ansonsten funktioniert garnichts. Jedes HTML-Element beginnt mit einem Start-Tag<> und einem End-Tag .



    Kapitel 2 Text im HTML bearbeiten

    Die folgenden Elemente sind zur Darstellung und Hervorhebung von Text und Zahlen gedacht.

    <h1>Das ist eine Überschrift<h1>
    <h2>Das ist eine Zwischenüberschrift<h2>
    <p>Das ist ein Absatz<p>
    <b>Alles was dazwischen steht wird fett geschrieben<b>
    <i>Alles was dazwischen steht wird kursiv geschrieben<i>
    <ins>Alles was dazwischen steht wird unterstrichen<ins>
    <del>Alles was dazwischen steht wird durchgestrichen<del>
    <sup>Alles was dazwischen steht wird hochgestellt<sup>
    <sub>Alles was dazwischen steht wird tiefgestellt<sub>

    <br /> Steht nach einem Absatz und ist ein Umbruch.
    <hr /> Steht vor oder nach einem Absatz und erzeugt eine horizontale graue Linie.




    Kapitel 3 Listen

    Geordnete Listen, Aufzählungszeichen Zahlen, erstellt man mit dem Element ol Listeninhalte erstellt man mit dem Element li.

    Ungeordnete Listen, Aufzählungszeichen Punkte werden mit dem ul Element erstellt mit dem Element li erstellt man Listeninhalte.

    Mit dem dl Element werden Definitionslisten erstellt.
    Im Element dt befindet sich der Begriff.
    Im Element dd die zugehörige Definition.




    Kapitel 4 Links

    Soll ein Link zu einer anderen Website führen, so muss dieser ein absoluter (kompletter) URL sein. Will man einen Link zu einer Seite auf derselben Seite erstellen, so verwendet man relative Pfade. Um einen Ordner zurückzukommen schreibt man in den relativen Pfad «../»



    Kapitel 5 Bilder

    Das leere Element <img> hat keinen Endtag und wird genutzt, um Bilder einzubetten. Das Attribut <src> wird angegeben, um die Herkunft des Bildes mitzuteilen.
    Das <alt> Attribut wird für eine kleine Beschreibung des Bildes genutzt, welche angezeigt wird, wenn das Bild nicht geladen wird und es wird von Screenreadern vorgelesen. Zusätzlich kann man das <title> Attribut nutzten, um weitere Infos anzuzeigen, während ein User über das Bild hovert.
    Das Element <figure> hält ein Bild und seine Unterschrift <figcaption> zusammen.




    Kapitel 6 Tabellen

    Um eine Tabelle zu erstellen braucht man das Element <table>. Um dann eine Zeile zu starten braucht man <tr>. Der eigentliche Inhalt der Tabelle wird mit <td> oder mit <th> für Überschriften eingefügt. Auch bei leeren Zellen muss das Element geschrieben werden. Mit colspan und rowspan können Zeilen und Spalten zusammengefügt werden.




    Kapitel 7 Formulare

    Das wahrscheinlich bekannteste Formular im Web ist das Suchfeld in der Mitte der Startseite von Google.


    Ein Formular wird im Body der HTML-Datei mit den Tags <form> und </form> definiert. Innerhalb einer HTML-Datei können auch mehrere Formulare definiert werden, die jedoch nicht ineinander verschachtelt sein dürfen. Andere HTML-Tags wie <table>, <img src="xyz.gif">, <a href="test.html"> lassen sich innerhalb des <form>-Tags problemlos einsetzen.



    Eingabefelder
    Mit dem HTML-Tag <input> lassen sich verschiedene Eingabefelder definieren. Es ist kein End-Tag </input> erforderlich. Mit dem Attribut type können die Felder für unterschiedliche Aktionen eingesetzt werden.

    Einzeilige Textfelder
    Für Textfelder stehen mehrere Attribute zur Verfügung, mit denen sie genau definiert werden können.

    Mehrzeilige Textfelder
    Auch für mehrzeilige Textfelder stehen mehrere Attribute zur Verfügung, über die das Textfeld genau definiert werden kann.

    Passwortfelder
    <input type="password"> Definiert ein Passwortfeld.

    Checkbuttons
    Checkbuttons ermöglichen dem Benutzer ein schnelles Aus- oder Abwählen eines Feldes. In der Regel werden mehrere Felder gruppiert, aus denen der Benutzer dann eine Auswahl treffen kann. In einer Gruppe von Checkbuttons ist es möglich, mehrere Elemente auszuwählen.

    Radiobuttons
    <input type="radio"> Definiert einen Radiobutton.

    Submit- und Resetbuttons
    Der Submitbutton überträgt die Formulardaten an den Server.
    Der Resetbutton ermöglicht es dem Benutzer, die eingegebenen Daten im Formular zu löschen.

    Auswahllisten (Drop-Down-Box)
    Darstellung und Aufbau einer Auswahlliste, Gültige Attribute für das <select>-Tag, Gültige Attribute für das <option>-Tag, Beispiel: Auswahllisten

    Versteckte Felder (hidden fields)
    Versteckte Felder werden vom Browser ignoriert, sie sind für den Benutzer unsichtbar. Sie kommen in Formularen zum Einsatz, um bestimmte Zusatzinformationen an den Server zu übertragen und an das CGI-Programm zu übergeben.





    Kapitel 8 Markup

    Mit DOCTYPE teilen Sie dem Browser mit, welche Version von HTML Sie verwenden.
    Zwischen den Markierungen <!- - und - -> können Sie Kommentare in den Code einfügen.
    Mit den Elementen <id> und <class> können Sie einzelne Elemente bezeichnen.
    Mit den Elementen <div> und <span> können Sie andere Elemente als Block bzw. inline gruppieren.
    Mit <iframes> betten Sie in Ihrer Webseite ein Fenster ein, indem andere Seiten angezeigt werden können.
    In dem Tag <meta> können Sie alle möglichen Arten von Informationen über Ihre Webseite angeben.
    Einige Sonderzeichen werden im HTML-Code verwendet und sind daher (reserviert z.B. die spitzen Klammern).

    Was ist CSS

    Das Cascading Style Sheets erlaubt uns Webseiten mit Farben, verschiedenen Schriften, Video, usw. kreativ zu gestalten. Das Konzept der Cascading Style Sheets wurde 1994 von Håkon Wium Lie vorgeschlagen. Bert Bos arbeitete zu dieser Zeit an der Entwicklung eines Darstellungsprogrammes namens Argo, der seine eigene Stilvorlagensprache benutzte. Die beiden entschieden sich zusammenzuarbeiten und CSS zu entwickeln.

    Es gab zu dieser Zeit auch andere Sprachen mit dem gleichen Ziel, CSS brachte aber als erste die Idee auf, Regeln zu definieren, die über mehrere Stilvorlagen hinweg vererbt werden konnten.

    Nach der Präsentation von CSS durch Håkon an der Konferenz „Mosaic and the Web“ in Chicago 1994, und später mit Bos 1995 wurde das World Wide Web Consortium (W3C) auf CSS aufmerksam. Håkon und Bos arbeiteten in diesem Rahmen an CSS weiter, zusammen mit anderen Mitgliedern, unter anderem Thomas Reardon von der Firma Microsoft. Im Dezember 1996 wurde dann die CSS Level 1 Recommendation publiziert. Diese wird in aktuellen Rendering-Engines vollständig umgesetzt


    Kapitel 2 Farbe

    Diese Befehle erlauben dem Benutzer die Farben von Texten und Hintergründen zu bearbeiten und zu verändern. Befehle die mit color anfangen beeinflussen die Textfarbe. Befehle die mit background-color beginnen beeinflussen die Hintergrundfarbe. Befehle die mit opacity beginnen beeinflussen die Deckkraft.


    color: blue; color: rgb(xxx, xxx, xxx) ; color: #66cdaa;

    background-color : blue ; background-color : rgb(xxx, xxx, xxx) ; background-color : #66cdaa;

    opacity : 50% ;

    Link: https://html-color-codes.info/webfarben_hexcodes/

    Kapitel 3 Text

    Schriftarten gibt man mit der Eigenschaft font-family und dem Eigenschaftswert «Name der Schrift» an. Die Grösse eines Textes kann man mit der Eigenschaft font-size angegeben werden. Mit der Eigenschaft font-weight kann man die dicke der Schriftart beeinflussen. Text b>italic, oblique oder normal anzeigen lassen kann man mit der Eigenschaft font-style. Mit der Eigenschaft text-decoration ist es möglich Text zu unterstreichen usw. Mit line-height kann man den Abstand zwischen Zeilen verändern. Die Eigenschaft text-align positioniert den Text. :hover, :active, :focus sind Pseudoklassen, welche im CSS bearbeiten werden können.



    Kapitel 4 Kästen

    Mit den Eigenschaften width und height kann man die Höhe und Breite eines Kastens verändern.

    Die Breite/Höhe einschränken kann man mit den Eigenschaften max-width/height und min-width/height.

    Falls man zu viel Inhalt für einen Kasten hat, kann man mit der Eigenschaft overflow und dem Eigenschaftswert hidden den zusätzlichen Inhalt ausblenden oder mit scroll einen scroll Effekt erzeugen.

    Mit der Eigenschaft border-width lässt sich die dicke der Border regeln, mit border-style die Art der Border-Linie und mit border-color die Farbe.

    Die Eigenschaftswerte von padding und margin werden mit vier Werten im Uhrzeigersinn bestimmt, gibt man nur zwei Werte ein regelt der erste top/bottom und der zweite left/right.

    Eine abgerundete Border kann man mit der Eigenschaft border-radius erstellen.








    Kapitel 5 Listen & Tabellen

    Mit der Eigenschaft list-style / -image kann man Listen anpassen, in dem man die Position und das Aussehen der Aufzählungspunkte verändert.

    Um Tabellen zu bearbeiten braucht man empty-cells damit kann man Zellen show oder hide, mit border-spacing kann man den Abstand zwischen den Zellen anpassen, zusätzlich kann man mit border-collapse die Rahmen zu einem einzigen zusammenfassen oder separieren.

    Falls man den Cursor des Benutzers anpassen will kann man die eigenschaft cursor nutzen.








    Kapitel 6 Layout

    Die Eigenschaft position hat vier Werte, static der Standard, relative mit welcher ein Element verschoben werden kann, mit dem absolut Wert wird ein Element komplett aus dem normalen Fluss genommen, und fixed was genutzt wird, wenn man ein Kasten an Ort und Stelle anheften will.

    Schwimmende Elemente erstellt man mit float, die restlichen Elemente umfliessen dann dieses. Der z-index ist eine Eigenschaft, die die «höhe» der Elemente bestimmt.

    Fals man ein CSS-File im einem CSS-Flie einbinden will kann man das Erreichen mit @import.



    Kapitel 7 Bilder

    Um Bilder in der Grösse zu verändern nutzt man die Eigenschaften width und height. Um ein Bild einzumitten muss man ihm display: block; und margin: 0px auto; sagen

    Um ein Hintergurndbild einzufügen nutzt man die Eigenschschaft background zuerst gibt man die background-color an, dann background-image, background-repeat, background-attachment und zuletzt background-position. Falls man etwas nicht ausfüllen will kann man es einfach auslassen.
